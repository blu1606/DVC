Lá chắn Bảo mật PII Shield (Local Zero-State)

Mọi số CCCD, Họ tên, Số điện thoại đều bị lọc, che giấu thành token bảo mật ngay tại thiết bị của người dùng. Dữ liệu thật chỉ được giải mã và điền ngược lại ô nhập liệu biểu mẫu khi có sự phê duyệt trực quan tại máy khách.

Đợi đồng bộ AJAX 3 cấp

Hướng dẫn cài đặt

Chỉ mất vài bước để triển khai thử nghiệm

EasyDVC gồm 2 thành phần chính: Tiện ích Chrome Extension và Máy chủ Agent Backend trung gian.

Bước 1: Tải mã nguồn & Đóng gói Chrome Extension

Mở terminal trên máy tính và chạy các lệnh sau để tải dự án từ GitHub Repository: 

git clone https://github.com/blu1606/DVC.git
cd DVC/extension
pnpm install
npx esbuild webrtc-client.js --bundle --outfile=webrtc-client.bundle.js

Bước 2: Nạp Extension vào Google Chrome

Nạp extension đã giải nén vào trình duyệt Chrome của bạn:

  1. Mở Chrome và truy cập địa chỉ chrome://extensions/
  2. Bật nút Chế độ nhà phát triển (Developer mode) ở góc trên bên phải.
  3. Chọn nút Tải tiện ích đã giải nén (Load unpacked) ở góc trái và chọn thư mục DVC/extension.

Bước 3: Khởi chạy Python Agent Server (Tùy chọn)

Để chạy đầy đủ tính năng hội thoại âm thanh qua WebRTC/WebSocket: 

cd ../backend
python -m venv .venv
# Trên Windows:
.venv\Scripts\activate
# Cài đặt thư viện & chạy server:
pip install -r requirements.txt
python agent_server.py
